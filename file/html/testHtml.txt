        &nbsp;&nbsp;&nbsp;&nbsp;张家港市委书记
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;姚林荣书记您好：
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;张家港市第一人民医院医务人员涉嫌故意杀人。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;所诉是造谣，你处理我。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;所诉是事实，你要处理。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;你不管是事实还是造谣你都不处理，我将此信发上网。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;我母亲陈梦珍患乳酸酸中毒（血酸过高）于2012年9月25日上午11.30点左右住进张家市第一人民医院重症室，2012年9月27日上午6点左右死亡。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;死亡原因是重症室医务人员用一硬物直接从嘴中插进陈梦珍左肺部（见照片1），插进同时病人因疼痛挣扎，医务人员强行按住造成右锁骨与肩钾骨接口处断开、错位（见照片2），硬物插进左肺部后在短时间内造成肺部大量积血，大面积病变，造成肺部严重损伤失血过多，病人于第二天死亡。再看入医院时拍摄的肺片（见照片3），入院时重症监护室医生拿到肺片时对我说从片子上看不出肺上有什么问题。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;上述行为如是医务人员在医疗过成中的过失行为造成病人死亡，还好说，但是，当医务人员拿到病人被硬物插进肺部的片子后（由医务人员过失造成的），没有对病人采取任何抢救措施，眼睁睁看着病人死去，这时事态的性质发生改变，这是故意杀人，应受到法律的制裁，同时张家港市第一人民医院也将承担重大责任，为什么将重症室这样重要的岗位，安排医术如此差、医德极差的医务人员？（拍摄硬物插进肺部的片子是2012年9月26日上午11时17分，病人死亡是2012年9月27日上午6时左右，看到硬物在肺部与死亡有19小时。）
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;在这19小时里医生没有对病人采取任何抢救措施，眼睁睁看着病人死去，这时事态的性质发生改变，这是故意杀人。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;2013年4月15日我向张家港市公安局报案，2013年4月20日警察对我说你要告的人是市刑警大队长的连襟（一担挑），2013年5月2日张家港市公安局作出不予立案通知书。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;姚林荣书记，铁证如山的犯罪证据，就因为是张家港市刑警大队长的连襟，就能妄法吗？
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;请求：姚林荣书记要为民做主。
       <br />
       <br />&nbsp;&nbsp;&nbsp;&nbsp;谢谢！！！

if (typeof HTMLElement != "undefined" && !HTMLElement.prototype.insertAdjacentElement) { 
    HTMLElement.prototype.insertAdjacentElement = function (where, parsedNode) { 
        switch (where) { 
            case "beforeBegin": 
                this.parentNode.insertBefore(parsedNode, this); 
                break; 
            case "afterBegin": 
                this.insertBefore(parsedNode, this.firstChild); 
                break; 
            case "beforeEnd": 
                this.appendChild(parsedNode); 
                break; 
            case "afterEnd": 
                if (this.nextSibling) { 
                    this.parentNode.insertBefore(parsedNode, this.nextSibling); 
                } else { 
                    this.parentNode.appendChild(parsedNode); 
                } 
                break; 
        } 
    }; 
    HTMLElement.prototype.insertAdjacentHTML = function (where, htmlStr) { 
        var r = this.ownerDocument.createRange(); 
        r.setStartBefore(this); 
        var parsedHTML = r.createContextualFragment(htmlStr); 
        this.insertAdjacentElement(where, parsedHTML); 
    }; 
    HTMLElement.prototype.insertAdjacentText = function (where, txtStr) { 
        var parsedText = document.createTextNode(txtStr); 
        this.insertAdjacentElement(where, parsedText); 
    }; 
}
